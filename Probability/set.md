### (3)

1. A = {1,2} B = {1,2,3,4} C ={3,4} D = {4,5,6,7}

- A ⊆ B --> `true`
- A = B --> `false`
- B ~ D --> `true`
- D ⊆ C --> `false`
- C ⊆ D --> `false`
- A ≠ B --> `true`
